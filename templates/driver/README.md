# Template: Driver

Este template cria um novo driver para o Redstone OS.

## Uso

```bash
anvil template new driver mydriver
```

## Arquivos Criados

- `drivers/{{name}}/Cargo.toml`
- `drivers/{{name}}/src/main.rs`
- `drivers/{{name}}/README.md`

## Variáveis

- `{{name}}` - Nome do driver
- `{{description}}` - Descrição do driver
- `{{author}}` - Autor
